Como conectar ao Wokwi
1
Abra o projeto no Wokwi e inicie a simulação clicando em ▶ Start Simulation.
2
No painel do Wokwi, clique em Network (ou veja o Serial Monitor) para obter o IP atribuído ao ESP32. O IP geralmente começa com 10.0.0. ou 192.168..
3
Cole o IP no campo acima e clique em Conectar. Este dashboard irá consultar /api/status a cada 2 segundos automaticamente.
4
Se estiver testando localmente sem o Wokwi, clique em Demo para ver os dados simulados em tempo real.
5
⚠️ Caso o navegador bloqueie a requisição por CORS, abra o Wokwi no mesmo navegador ou use uma extensão como CORS Unblock. O firmware já envia os headers Access-Control-Allow-Origin: *.
